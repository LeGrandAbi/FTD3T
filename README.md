# FTD3T
A Pirate Themed Minecraft Modpack

I - Installation de serveur:

1/ cloner le git
2/ faire une fresh install de server forge 1.20.1 a cet emplacement
3/ lancer le serveur une fois pour generer les eula et le minimum
4/ accepter les eula
5/ telecharger, extraire et mettre les mods dans le dossier eponyme
6/ lancer le serveur

II - Installation de client:

1/ installer une instance forge 1.20.1 sur votre launcher
2/ lancer une premiere fois votre instance
3/ telecharger, extraire et mettre les mods dans le dossier eponyme
4/ telecharger et mettre le ressourcepack dans le dossier eponyme
